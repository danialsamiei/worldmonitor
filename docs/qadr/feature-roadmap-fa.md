# نقشه توسعه قابلیت‌های پیشرفته QADR110

## ۱) پدافند جنگ شناختی
- تشخیص عملیات نفوذ شناختی و دستکاری روایت
- تحلیل شبکه انتشار روایت در پلتفرم‌ها
- شاخص «شدت جنگ شناختی» با هشدار لحظه‌ای

## ۲) مدیریت بحران، آشوب و جرایم (منطبق با حقوق بشر)
- داشبورد هشدار چندشاخصه
- پیشنهاد اقدام مرحله‌ای با قیدهای حقوق بشری
- ثبت تصمیمات و ممیزی (Audit Trail)

## ۳) رصد جنگ رسانه‌ای و گروه‌بندی‌های بین‌المللی
- North/South, G8, NATO, EU, OPEC, BRICS, NAM, GCC, OIC
- تحلیل تطبیقی مواضع رسمی و روایت رسانه‌ای

## ۴) تحلیل شبکه‌های اجتماعی
- رصد ترندهای Google/X/Instagram/TikTok/Facebook
- تحلیل کمی (حجم/سرعت/تکرار) + کیفی (چارچوب‌بندی/سوگیری)
- امتیاز اصالت/اعتبار منبع

## ۵) اتصال AI چندمدلی
- OpenRouter به‌عنوان مسیردهی اصلی
- Ollama/vLLM به‌عنوان پردازش محلی
- تعریف پروفایل تحلیل سریع/عمیق/بحران/سیاستی

## ۶) BotOps
- Telegram Bot برای هشدار و فرمان
- Bale Bot برای اطلاع‌رسانی داخلی
- قالب گزارش استاندارد برای کانال‌ها

## ۷) Workbooks و Notebooks
- ذخیره تحلیل‌ها در workbook/notebook
- نسخه‌بندی گزارش‌ها
- خروجی قابل اشتراک‌گذاری و مخزن‌پذیر

## ۸) رصد دفاعی DarkWeb/DarkNet (قانونی)
- تحلیل تهدید و نشت داده از منابع عمومی و قانونی
- ارزیابی ریسک بدون ارائه روش دسترسی غیرمجاز
- پایش موتورهای جستجو/دایرکتوری‌های پژوهشی در چارچوب سیاست‌های حقوقی

## ۹) رسانه‌های ایران + آپارات/تلوبیون
- ماتریس درون‌مرزی/برون‌مرزی/مستقل/اپوزیسیون
- تحلیل شکاف روایت بین طیف‌های رسانه‌ای

## ۱۰) داده‌های زیرساخت، ترافیک و سایبر
- ترافیک جاده‌ای/هوایی/دریایی (semi-realtime)
- داده‌های cyber attack و سلامت IXPها
- همجوشی GDELT + NetBlocks + Google Trends

## ۱۱) طراحی 2D Model + Pipeline Workflow
- خط لوله ingest → normalize → score → alert → report
- اجرای شبه‌بلادرنگ با بازه‌های 30s/60s/5m

## ۱۲) پایپلاین رسانه‌ای ایران/اسرائیل در شبکه‌های اجتماعی
- پایپلاین‌های تفکیکی حکومتی/مستقل/اپوزیسیون
- پوشش Telegram, Instagram, X, Web
- تحلیل همزمان روایت، تناقض، و تغییرات framing
